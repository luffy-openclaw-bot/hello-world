# 🎬 Demo 演示腳本

**話術：**

> - 實時 Dashboard

#### Step 2: GPS 打卡
```
1. 系統自動獲取 GPS 位置
2. 顯示經緯度

```

**話術：**

**話術：**

### Part 5: Q&A (1 分鐘)

**常見問題準備：**

**Q1: 點解唔用 App？**
> A: Web App 無需安裝，更新方便，成本低，
>    而且而家瀏覽器嘅 GPS 功能已經好成熟。

**Q2: 準確度點樣？**
> A: 瀏覽器 GPS 準確度約 10-50 米，
>    我哋設置咗合理嘅誤差範圍（100-200 米）。

**Q3: 點防止作弊？**
> A: 我哋有三大防作弊機制：
>    1. GPS 欺騙檢測
>    2. 時間異常檢測
>    3. 設備指紋
>    再加上可選嘅拍照驗證。

**Q4: 幾錢一個月？**
> A: 根據報價，首年 $6,000（包括開發同維護），
>    第二年開始 $2,800/年，平均每月$233，
>    比傳統考勤系統平好多。
