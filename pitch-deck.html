🚨 問題陳述

當前挑戰：外勤員工考勤追蹤人手操作且不可靠

痛點

  • 無法實時查看員工工作期間位置
  • 難以核實早退或遲到
  • 現有承辦商方案要求最低人數 (10-20 人)
  • 原生 App 開發成本高且難維護

業務影響

  • 薪資處理效率低下
  • 潛在時間欺詐
  • 管理開支增加

💡 解決方案概述

網頁版 GPS 打卡系統

核心概念：員工通過手機瀏覽器打卡，無需安裝 App

主要組件

  • 手機友好打卡介面 + GPS + 自拍驗證
  • 管理員 Dashboard 實時監控 + 報告生成
  • 基本防欺詐機制確保數據完整性 (Optional)
  • 單一組織內部系統部署 (非 SaaS)

🏗️ 技術架構

前端

  • React/Vue + Vite
  • Leaflet.js 地圖
  • PWA 離線支援

後端

  • Node.js/Express 或 PHP/Laravel
  • JWT 認證

數據庫

  • MySQL/PostgreSQL
  • 或 Supabase/Firebase 免費層

部署

  • 部署到現有網站主機
  • Cloudflare DNS/SSL

⚠️ 系統限制和範圍

  • 部署：僅內部使用，單一組織，無第三方服務提供
  • 用戶限制：最多 100 名員工賬戶，5 個管理員賬戶
  • 存儲限制：5GB 圖片文件，1GB 數據庫記錄
  • 訪問：單域名部署，無多租戶，無外部 API 暴露
  • 許可：不可轉讓，無白標或轉售權利
  • 維護範圍：僅錯誤修復，不包含功能擴展

⚖️ 比較：自建 vs 購買 vs SaaS

選項 A：定制 Web App (本提案) ✅ 推薦

優點：最低成本 • 完全控制 • 無經常性許可 • 快速部署

缺點：有限防欺詐 • 網頁平台限制 • 客戶管理合規

選項 B：商業 SaaS (如 JustLogin, Hubstaff)

優點：成熟功能 • 強大防欺詐 • 供應商處理合規

缺點：經常性按用戶費用 • 最低人數限制 • 定制較少

選項 C：原生 App 開發

優點：最佳防欺詐 • 背景追蹤 • 豐富 UX

缺點：高開發成本 (5-8 倍) • App Store 審批 • 雙平台維護

❓ FAQ 常見問題

Q: 員工可以虛擬定位嗎？

A: Web App 有基本驗證但無法檢測高級欺騙；建議結合管理檢查 (如隨機照片審查/主管驗證)

Q: 員工無信號怎麼辦？

A: PWA 離線支援緩存本地數據，連接恢復時同步；

但照片上傳可能會丟失

Q: 誰擁有源代碼？

A: 客戶獲得內部使用許可；轉售或白標需單獨商業協議

Q: 可以後續添加功能嗎？

A: 可以，新功能單獨報價；基礎維護僅涵蓋錯誤修復

Q: 員工位置 24/7 追蹤嗎？

A: 不，位置僅在打卡瞬間捕捉；無背景追蹤
